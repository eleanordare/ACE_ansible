# ACE_ansible

installs:
- JDK 8
- apache maven
- git
- apache tomcat
- mongodb
- npm
- nodejs
- rubygems for openshift

to run:
- `yum install ansible` in Terminal
- `git clone` this repository to your desktop
- navigate to cloned ACE_ansible repository
- run `ansible-playbook playbook.yml` from inside ACE_ansible
